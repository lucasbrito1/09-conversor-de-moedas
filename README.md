### 🌟 Features

- Layout responsivo
- Conversão funcionando
- Uso de API para obter cotação atual
- Gráfico com o histórico de cotações (Somente layout)

### 🚀 Tecnologias utilizadas

- React JS
- TypeScript
- Tailwind CSS
